Intro to Javascript

Javascript is one of the most popular programmming languages in the world. Originally used only in the web, however, thanks to Node.JS it is able to run outside the browser.

Basic Definition of Javascript

Javascript is a high-level programming language. It is a dynamic programming language unlike a static one. Javascript is a dynamically typed programming language and lastly it is interpreted.

Variables

A variable is a literal assignment to an identifier Javascript variables don't have a type attached.

Using let vs const

Intoduced in the ES2, let is a term to just assign variables. const means that the variable assigned cannot be changed similar to final in Java. varaibles declared using let can be altered and reassigned however, const cannot.

Types

You might sometimes read that javascript is untyped, but thats wrong. It's true that you can assign all sorts of diffrent types to a variabl, but JS has types.

Primitive Types

Primitive Types
Primitive types are Number, Strings, and Booleans. There are two special types Null and Undefined.
Numbers
Unlike Other languages Numbers include one type called floats.
Strings
A string is a sequence of characters enclosed with quotes
Booleans
JS reserves two words for booleans true and false
Null
Null is a special value that indicates the absence of a value
Undefined
Undefined indicates that the variable has not been initalized and the value is absent
